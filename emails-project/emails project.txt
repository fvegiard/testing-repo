## üîß Technical Details

 

### PowerShell Commands Used:

```powershell

# Move invoice PDFs matching patterns

$sourceDir = "C:\Users\DRsac.DRELECTRIQUE\Downloads"

$destDir = "C:\Users\DRsac.DRELECTRIQUE\dev\claude\emails"

Get-ChildItem -Path $sourceDir -Filter "*.pdf" |

Where-Object { $_.Name -match "^(fact|Fact|facture|Facture|IN\d+|Inv\d+|S-\d+|doc\d+)" } |

ForEach-Object {

    Move-Item -Path $_.FullName -Destination $destDir -Force

    Write-Host "Moved: $($_.Name)"

}

```

 

### File Matching Patterns:

- `fact*` or `Fact*` - French invoices (factures)

- `facture*` or `Facture*` - French invoices

- `IN\d+` - Invoice numbers starting with IN

- `Inv\d+` - Invoice numbers starting with Inv

- `S-\d+` - Service order documents

- `doc\d+` - Scanned documents with date stamps
